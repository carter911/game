2 New Order

POST /newOrder

Parameters:

merchant_id :unique order number from your system
login :EA e-mail (string)
password : EA password (string)
amount : Amount to fill in to account (int)
platform :ps4 (string)
backup1 - backup code 1 (string)
backup2 - backup code 2 (string)
backup3 - backup code 3 (string)
example
{user: token, merchant_id: 123123, login: 'test@test.com', 'password': 'password', 'amount': 1000, 'platform': 'ps4', 'backup1': '123456', 'backup2': '123456', 'backup3': '123456'}

Response:

{"data":{"id":"20","time":1571546267,"status":"ORDER CREATE"},"code":200,"message":"create order success"}

Status List

captcha-can’t solve captcha
tradepilefull-there are no space no trade pileto move cards
toomanyaction-account is temporary banned by ea
unexpected-technical issue on our site
<200 coins-less than 200 coins in account
wrongbackup-wrong back up codes
wronglogin-login or password is wrong
marketlocked-webapp market is locked
csmonline-customer online
nologinverification-its when customer need to login to webapp and use code from mail,then make new back ups when its working
transferring-account info is correct, transfer in progress
checking info-supplier is checking origin account info,waiting for delivery
end-order finished
new-new order not yet checked
